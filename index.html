Pacman






You can build your own level by using the following symbols:

  • # : wall
  • g : ghost
  • p : pacman
  • o : point
  • x : empty nonsolid space

Rules:

  • Level is limited by walls
  • Every non-wall element is reachable
  • Level contains at least one point, pacman and ghost


Controls

  • WASD
  • ArrowKeys
